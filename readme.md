# 📋 Gestor de Tareas y Formación

Aplicación web desarrollada en **Python** que permite gestionar **tareas** y **formación** desde una única interfaz.  
El objetivo del proyecto es facilitar la organización del trabajo y el seguimiento del aprendizaje o capacitación dentro de un mismo sistema.

---
cloudflared tunnel --url http://localhost:5000

## 🚀 Características

### ✅ Gestor de Tareas
- Crear nuevas tareas
- Editar tareas existentes
- Marcar tareas como completadas
- Eliminar tareas
- Visualizar el estado de las tareas

### 🎓 Gestor de Formación
- Registrar formaciones o cursos
- Añadir información relevante (nombre, descripción, estado, etc.)
- Seguimiento del progreso formativo
- Organización de la formación dentro de la aplicación
